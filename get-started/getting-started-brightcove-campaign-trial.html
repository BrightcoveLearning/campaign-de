Brightcove Campaign wurde für Vermarkter der Nachfragegenerierung entwickelt und erleichtert das Konfigurieren, Veröffentlichen und Messen der Videoleistung innerhalb von Kampagnen.

Eröffnen Sie ein Testkonto

Brightcove bietet Testkonten für die Brightcove-Kampagne an. Melden Sie sich an Hier.

Testkonten haben die folgenden Einschränkungen:

  • Es können maximal fünf Videos auf ein Konto hochgeladen werden. Wenn Sie mehr als 5 Videos hochladen müssen, während sich das Konto im Testmodus befindet, wenden Sie sich an unser Verkaufsteam
  • Es können maximal 3 Benutzer erstellt werden
  • Es kann nur eine Verbindung zu einer Marketing-Automatisierungsplattform hergestellt werden
