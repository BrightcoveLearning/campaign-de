Fahren von Videoansichten per E-Mail

{{ page.description }}

Die Veröffentlichungsfunktion in Brightcove Campaign™ kann verwendet werden, um auf einfache Weise den Einbettungscode und die Bilder zu generieren, die zum Verteilen eines Videos über eine Landingpage und E-Mail erforderlich sind. In der Regel wird der Video-Einbettungscode in eine Zielseite eingefügt und anschließend ein Link zur Zielseite per E-Mail gesendet. Der in diesem Thema beschriebene Prozess eignet sich am besten für eine Marketing Automation Platform (MAP) nicht verwendet werden. Wenn Sie einen MAP verwenden, empfiehlt Brightcove die Verwendung des regulären MAP Veröffentlichungsprozess.

Landing Page erstellen

Eine Zielseite ist eine eigenständige Webseite, die als Teil einer Marketingkampagne zur Bewerbung eines Produkts oder einer Dienstleistung verwendet werden kann. Das Hinzufügen von Videos zu Zielseiten ist eine großartige Möglichkeit, das Engagement der Zuschauer zu erhöhen und die Zuschauer zu Käufern zu machen. Es gibt eine Reihe von Anbietern, die die Möglichkeit bieten, Zielseiten zu erstellen. Zielseiten können auch mit der Software der Marketing-Automatisierungsplattform oder manuell mit benutzerdefiniertem HTML erstellt werden, wenn Sie über Entwicklerressourcen verfügen. Sobald Sie eine Zielseite erstellt haben, können Sie mit Brightcove Campaign ein Video in die Zielseite einbetten.

Einbettungscode generieren

Der Einbettungscode ist erforderlich, damit ein Video auf einer Zielseite abgespielt werden kann. Gehen Sie folgendermaßen vor, um den Einbettungscode für ein Video zu kopieren:

  1. Öffne das Videos Seite.
  2. Wählen Sie ein Video aus, indem Sie auf das Kontrollkästchen neben dem Video klicken, das Sie verwenden möchten.
  3. Klicken Sie auf Veröffentlichen.
  4. Kopiere das Code einbetten und fügen Sie es in Ihre Zielseite ein, auf der das Video angezeigt werden soll.
  5. Speichern Sie die Änderungen auf Ihrer Zielseite.
  6. Veröffentlichen Sie die Zielseite und stellen Sie sicher, dass Sie eine URL haben, unter der die Zielseite angezeigt werden kann.

Kopieren des Posterbildes

Auf den meisten Plattformen ist das direkte Einbetten von Videos in E-Mails’nicht möglich. Dies ist keine Brightcove-Einschränkung, fast jedes moderne E-Mail-System wird aus Sicherheitsgründen den Einbettungscode des Videoplayers entfernen. Wir können jedoch das Bewusstsein und den Traffic für Inhalte steigern, indem wir eine E-Mail mit etwas Teasertext und einem Posterbild erstellen. Das Posterbild sieht aus wie ein Videoplayer und ist ein anklickbarer Link, der zur Zielseite weiterleitet.

Gehen Sie folgendermaßen vor, um das Posterbild für ein Video zu kopieren:

  1. Drücke den Email Registerkarte oben auf der Seite.
  2. Klicken Sie auf Kopieren, um das Posterbild zu kopieren.
  3. Fügen Sie das Bild in das Tool ein, das die E-Mail senden soll.
  4. Erstellen Sie einen Hyperlink vom Bild zur Landingpage. Der Vorgang hierfür hängt vom verwendeten E-Mail-Programm ab.

Wenn die E-Mail gesendet wird, sehen die Leser das Video als Teil der E-Mail. Wenn sie auf das Bild klicken, werden sie zur Zielseite weitergeleitet, auf der sie das Video ansehen können.
