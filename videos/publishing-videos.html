Veröffentlichen von Videos

{{ page.description }}

Die Veröffentlichungsfunktion in Brightcove Campaign™ kann verwendet werden, um auf einfache Weise den Einbettungscode und die Bilder zu generieren, die für die Verbreitung eines Videos über eine Marketing Automation Platform (MAP) erforderlich sind. In der Regel wird der Einbettungscode in eine HTML-Landingpage eingefügt, die in einer MAP erstellt wurde, und dann wird ein Link zur Landingpage per E-Mail gesendet.

Veröffentlichen auf einer Webseite

Gehen Sie folgendermaßen vor, um den Einbettungscode für ein Video zu kopieren:

  1. Öffne das Videos Seite.
  2. Wählen Sie ein Video aus, indem Sie auf das Kontrollkästchen neben dem Video klicken.
  3. Klicken Sie auf Veröffentlichen.
  4. (Optional) Select a MAP Connection. Die Dropdown-Liste enthält eine Liste aller MAP-Verbindungen die erstellt wurden. Durch Auswahl einer MAP-Verbindung werden Videoanzeigeaktivitäten an die ausgewählte Plattform gesendet. Die Verfügbarkeit dieser Option basiert auf dem verwendeten Brightcove Campaign-Paket.
  5. (Optional) Select a Lead Form. Die Dropdown-Liste enthält eine Liste aller Bleiformulare die erstellt wurden. Durch Auswahl einer MAP-Verbindung kann während der Videowiedergabe ein Lead-Formular angezeigt werden, um Viewer-Informationen zu erfassen. Die Verfügbarkeit dieser Option basiert auf dem verwendeten Brightcove Campaign-Paket.
  6. Kopieren Sie den Einbettungscode und fügen Sie ihn in Ihre Landing Page ein. Dies ist der Code, der benötigt wird, um Ihr Video auf einer Webseite zu veröffentlichen.
  7. Kopiere das Video SEO Code und fügen Sie den Code zwischen dem <head> und </head> Tags auf Ihrer Zielseite. Der Video-SEO-Code verbessert die Sichtbarkeit der Zielseite für Suchmaschinen. Nach der Veröffentlichung Ihrer Seite bietet Google eine URL-Inspektionstool Hiermit können Sie detaillierte Google Index-Informationen zu einer URL auf Ihrer Website anzeigen.
  8. Kopiere das Facebook Tags und fügen Sie den Code zwischen dem <head> und </head> Tags auf Ihrer Zielseite. Die Facebook-Tags sind die Facebook Open Graph-Tags, mit denen Sie steuern können, was angezeigt wird, wenn die Seite für Facebook freigegeben wird.

Kopieren des Posterbildes

Auf den meisten Plattformen ist das direkte Einbetten von Videos in E-Mails’nicht möglich. Dies ist keine Brightcove-Einschränkung, fast jedes moderne E-Mail-System wird aus Sicherheitsgründen den Einbettungscode des Videoplayers entfernen. Wir können jedoch das Bewusstsein und den Traffic für Inhalte steigern, indem wir eine E-Mail mit etwas Teasertext und einem Posterbild erstellen. Das Posterbild sieht aus wie ein Videoplayer und ist ein anklickbarer Link, der auf die Video-Zielseite umleitet.

Gehen Sie folgendermaßen vor, um das Posterbild für ein Video zu kopieren:

  1. Drücke den Email Registerkarte oben auf der Seite.
  2. Klicken Sie auf Kopieren, um das Posterbild zu kopieren.
  3. Fügen Sie das Bild in das Tool ein, das die E-Mail senden soll.
  4. Erstellen Sie mit dem veröffentlichten Video einen Hyperlink vom Bild zur Zielseite.
