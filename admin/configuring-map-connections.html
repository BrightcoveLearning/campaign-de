Konfigurieren von Verbindungen zur Marketing-Automatisierung

{{ page.description }}

Brightcove Campaign™ kann so konfiguriert werden, dass sie in beliebte Marketing-Automatisierungsplattformen (MAP) wie HubSpot, Eloqua und Marketo integriert werden. Wenn die MAP-Integration konfiguriert ist, werden Videointeraktionsdaten für Videos, die mit Brightcove Campaign veröffentlicht wurden, mit der ausgewählten Marketing-Automatisierungsplattform synchronisiert.

Hinweis: Sie müssen ein Kontoadministrator sein, um auf das Administrationsmenü zugreifen zu können.
Hinweis: Der Verbindungsprozess öffnet Popup-Fenster in Ihrem Browser. Sie sollten alle Browser-Popup-Blocker deaktivieren, bevor Sie die Verbindung einrichten.

Gehen Sie folgendermaßen vor, um eine Integration mit einer Marketing-Automatisierungsplattform zu konfigurieren:

  1. Melden Sie sich bei Brightcove Campaign an.
  2. Klicken Sie in der Navigationskopfzeile auf das Zahnradsymbol () und dann MAP Connections. Eine Liste der aktuellen Verbindungen (falls vorhanden) wird angezeigt.
  3. Klicken Sie auf Neue Verbindung hinzufügen.
  4. Wählen Sie den Verbindungstyp aus. Informationen zum Herstellen einer Verbindung mit dieser Plattform finden Sie im entsprechenden Thema.
    • Eloqua
    • Marketo (REST API oder Munchkin)
    • HubSpot (REST-API oder clientseitige API)
    • Salesforce
