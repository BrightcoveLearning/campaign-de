Kontoeinstellungen verwalten

{{ page.description }}

So greifen Sie auf die Brightcove-Kampagne zu™ Kontoeinstellungen, klicken Sie auf das Zahnradsymbol () in the navigation header.

Hinweis: Sie müssen ein Kontoadministrator sein, um auf das Verwaltungsmenü zugreifen zu können.

Das Kontoeinstellungsmenü bietet die folgenden Optionen:

  • Kontoinformationen
  • Benutzerverwaltung
  • Account-Einstellungen
  • Medien-Sharing
  • Google Analytics
  • Adobe Analytics
  • MAP-Verbindungen
  • Bleiformulare
  • Player-Styling
  • Abrechnung

Kontoinformationen

Mit der Option Kontoinformationen können Sie den Kontonamen und die Postanschrift bearbeiten. Weitere Informationen finden Sie unter Kontoinformationen verwalten.

Benutzerverwaltung

Mit der Option Benutzerverwaltung können Sie die Benutzer verwalten, die zur Verwendung der Brightcove Campaign-Anwendung berechtigt sind. Weitere Informationen finden Sie unter Benutzer verwalten.

Account-Einstellungen

In den Kontoeinstellungen können Sie die Standardberichtszeitzone für das Konto festlegen und das Feld Videoinhaltstyp aktivieren. Die Berichtszeitzone wird verwendet, um Tagesgrenzen für die Berichterstellung von Daten zu berechnen. Die Änderung der Zeitzoneneinstellung für die Berichterstellung wirkt sich nur auf die künftigen Daten aus, und Änderungen werden nicht rückwirkend angewendet. Wenn Sie diese Einstellung ändern, sehen Sie möglicherweise eine flache Stelle oder einen Anstieg Ihrer Analysedaten, wenn Sie sich Tagesgrenzen ansehen, an denen die Änderung angewendet wurde. Aktualisierungen dieser Einstellung werden möglicherweise nicht sofort wirksam, und Analysedaten beziehen sich möglicherweise weiterhin auf die alte Einstellung, bis das System die Änderungen verarbeitet.

Durch Aktivieren der Videoinhaltstyp Vor Ort kann Brightcove Ihre Analysen mit denen ähnlicher Videos kompilieren und robustere Analysen bereitstellen, die den Video Benchmark Score (VBS) von Brightcove und individuelle Ratschläge enthalten.

Medien-Sharing

Mit der Option Medienfreigabe können Sie Videos einmal hochladen und die Videos dann für andere Brightcove-Konten freigeben. Weitere Informationen finden Sie unter Verwalten der Einstellungen für die Medienfreigabe.

Google Analytics

Mit der Option "Google Analytics" können Spielerereignisse und benutzerdefinierte Dimensionsdaten zu Berichtszwecken an ein Google Analytics-Konto gesendet werden. Weitere Informationen finden Sie unter Konfigurieren der Google Analytics-Ereignisverfolgung.

Adobe Analytics

Mit der Option Adobe Analytics können Ereignisverfolgungsdaten zu Berichtszwecken an ein Adobe Analytics-Konto gesendet werden. Weitere Informationen finden Sie unter Konfigurieren der Adobe Analytics-Ereignisverfolgung.

MAP-Verbindungen

Mit der Option MAP-Verbindungen können Sie Integrationen mit Marketing-Automatisierungsplattformen wie HubSpot, Eloqua und Marketo konfigurieren. Weitere Informationen finden Sie unter Konfigurieren von Marketing Automation Platform-Verbindungen.

Bleiformulare

Mit der Option Lead Forms können Sie Lead Forms erstellen, die während der Videowiedergabe angezeigt werden können. Weitere Informationen finden Sie unter Lead-Formulare erstellen.

Spieler-Styling

Mit der Option "Player-Styling" können Sie das Erscheinungsbild des Players, die Farben und eine Player-Überlagerung anpassen. Weitere Informationen finden Sie unter Styling-Spieler.

Abrechnung

Mit der Option Abrechnung können Sie die Zahlungsinformationen für das Konto anzeigen. Weitere Informationen finden Sie unter Anzeigen von Rechnungsinformationen.
