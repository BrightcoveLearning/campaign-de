Anzeigen von Rechnungsinformationen

{{ page.description }}

Administratoren können die Rechnungsinformationen für ein Konto anzeigen. Beachten Sie, dass Sie für den Zugriff auf die Abrechnungsseite ein Benutzer mit der Administratorrolle sein müssen und Ihrem Konto ein Jahresvertrag zugeordnet sein muss. So greifen Sie auf die Abrechnungsseite zu:

  1. Melden Sie sich bei Brightcove Campaign an™.
  2. Klicken Sie in der Navigationskopfzeile auf das Zahnradsymbol () and then Billing.

Die Hauptabrechnungsseite wird angezeigt. Das Abrechnung & Konto Dropdown kann verwendet werden, um ein anderes zu verwaltendes Konto auszuwählen.

Die Abrechnungsseite ist in verschiedene Abschnitte unterteilt, die im Folgenden behandelt werden.

Planen Sie Details

Das PLANDETAILS In diesem Abschnitt werden Informationen zu Ihrem aktuellen Plan aufgeführt, darunter:

  • Plan Name
  • Maximale Benutzerkonten
  • Maximale Videos im Konto
  • Ablaufdatum planen

Rechnungsinformation

Das RECHNUNGSINFORMATION Im Abschnitt werden die Rechnungsinformationen für das Girokonto angezeigt. Klicken Abrechnungsmanager um Ihre Zahlungsinformationen anzuzeigen / zu bearbeiten.

Nutzungsübersicht

Das ÜBERBLICK ÜBER DIE NUTZUNG Im Abschnitt werden Informationen zur Anzahl der erstellten Benutzerkonten und zur Anzahl der Videos im Konto angezeigt.

Kontakt

Das KONTAKT In diesem Bereich können Sie Brightcove eine E-Mail mit Fragen zur Rechnungsstellung senden.
